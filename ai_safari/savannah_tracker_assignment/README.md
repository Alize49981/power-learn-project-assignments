# 🧠 AfyaTech Precision Prompting Challenge

## Overview
This project was completed as part of the **Precision Prompting Challenge (Savannah Framework)**.  
It focuses on improving AI prompts for an SMS-based maternal health assistant targeting rural Kenya and Uganda.

The goal is to eliminate urban bias and ensure culturally, economically, and geographically relevant AI outputs.

---

## 🚨 Problem
Initial AI prompts produced:
- Urban-centered health advice
- Non-local food recommendations
- Ignored rural transport constraints
- Lack of cultural and economic awareness

---

## 🛠️ Solution Approach
We applied:
- AIM Framework (Audience, Instruction, Modality)
- MAP Framework (Memory, Action, Personalization)
- Chain-of-Thought reasoning
- Verifier Pattern for safety-critical outputs

## 🌍 Key Focus Areas
